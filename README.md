## #! WHAT IS THIS

Top-down arena shooter built with raw JavaScript & Canvas API. No frameworks. No BS.

Wave-based combat // Progressive difficulty // Three enemy AI types // Power-up madness

## $$ THE ARSENAL

- **WASD / ARROWS** » move your ass
- **MOUSE CLICK** » shoot everything
- **P / ESC** » pause (coward mode)
- **R** » reset when you fail

## !? ENEMY TYPES

All look the same. All want you dead.

- **DUMB** » walks straight at you like an idiot
- **SMART** (yellow dot) » dodges & shoots back
- **TELEPORT** (cyan dot) » fades out & reappears in your face

## % POWER-UPS

Yellow squares = health restoration  
Magenta squares = triple-shot chaos for 5 seconds
